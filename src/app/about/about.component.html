About this project

This project is a result of the Angular assignment from Better. It showcases a simple application that displays a list of doctors and their tasks. The application was generated with Angular CLI version 17.3.1. and was built using Angular and Angular Material. The data for the doctors and tasks is fetched from the JSONPlaceholder API.

  • Doctors: Get a list of doctors by visiting https://jsonplaceholder.typicode.com/users.
  • Tasks: To see all tasks, head to https://jsonplaceholder.typicode.com/todos .
  • Tasks for a Specific Doctor: If you're interested in tasks for a particular doctor (for example, with doctor ID 1), you can find them at https://jsonplaceholder.typicode.com/users/1/todos.

Tests are written using Jasmine and Karma.

The source code is accessible on GitHub.
